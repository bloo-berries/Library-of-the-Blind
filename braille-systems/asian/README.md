# Asian Braille Systems

## Major Asian Braille Codes

### Arabic Braille
- **Languages Supported:** Arabic, Persian (Farsi), Urdu
- **Key Characteristics:** Right-to-left orientation, contextual shaping
- **Status:** Active

### Hebrew Braille
- **Languages Supported:** Hebrew, Yiddish
- **Key Characteristics:** Uses same cell patterns as English but mapped to Hebrew letters
- **Status:** Active

### Japanese Tenji (Braille)
- **Languages Supported:** Japanese kana (hiragana/katakana) and kanji (via "kanji Braille")
- **Key Characteristics:** Two-cell system for kanji radicals
- **Status:** Active

### Korean Hangul Braille
- **Languages Supported:** Korean Hangul
- **Key Characteristics:** 24-cell base, combines jamo
- **Status:** Active

### Chinese Braille
- **Languages Supported:** Mandarin (Pinyin-based), Cantonese (Jyutping)
- **Key Characteristics:** Two-cell system representing phonetics
- **Status:** Active

### Indian Braille (Bharati Braille)
- **Languages Supported:** Hindi, Bengali, Tamil, Telugu, Malayalam, Gujarati, Punjabi, Assamese, Oriya, Kannada, Sanskrit, etc.
- **Key Characteristics:** Unified code across 22 Indian languages
- **Status:** Active

### Thai Braille
- **Languages Supported:** Thai
- **Key Characteristics:** Adapted from English Braille with tone markers
- **Status:** Active

### Vietnamese Braille
- **Languages Supported:** Vietnamese
- **Key Characteristics:** Uses diacritic cells for tonal marks
- **Status:** Active

### Malay/Indonesian Braille
- **Languages Supported:** Malay, Indonesian
- **Key Characteristics:** Derived from English Braille with local letters
- **Status:** Active

### Mongolian Braille
- **Languages Supported:** Mongolian (Cyrillic script)
- **Key Characteristics:** Based on Russian Braille
- **Status:** Active

### Georgian Braille
- **Languages Supported:** Georgian
- **Key Characteristics:** Unique mapping
- **Status:** Active

### Armenian Braille
- **Languages Supported:** Armenian
- **Key Characteristics:** Distinct cell assignments
- **Status:** Active

### Tibetan Braille
- **Languages Supported:** Tibetan
- **Key Characteristics:** Small community usage
- **Status:** Active
